# LogLens++ CLI Usage Guide

The LogLens++ CLI provides a powerful command-line interface for log analytics. This guide demonstrates all available commands.

## Installation

After installing the package, the `loglens` command is available:

```bash
pip install -e .
loglens --help
```

## Commands

### 1. Ingest Logs

Ingest logs from a file into the database.

```bash
# Ingest JSON logs
loglens ingest app.log --format json

# Ingest plain text logs
loglens ingest app.log --format text

# Auto-detect format
loglens ingest app.log

# Specify default source
loglens ingest app.log --source myapp

# Use custom database
loglens ingest app.log --db production.db
```

**Example:**
```bash
$ loglens ingest /var/log/app.log --format json
Ingesting logs from: /var/log/app.log
âœ“ Ingested 1234 events into loglens.db
```

### 2. Query Metrics

List and view stored metrics.

```bash
# List all metrics
loglens metrics list

# Show specific metric
loglens metrics show error_count

# Show with limit
loglens metrics show error_count --limit 20

# Use custom database
loglens metrics list --db production.db
```

### 3. SQL Queries

Execute SQL queries against the database.

```bash
# Simple query
loglens query "SELECT * FROM events LIMIT 10"

# Aggregation query
loglens query "SELECT level, COUNT(*) FROM events GROUP BY level"

# Time-based query
loglens query "SELECT * FROM events WHERE timestamp > '2024-01-01'"

# JSON output
loglens query "SELECT * FROM metrics" --format json

# Use custom database
loglens query "SELECT * FROM events" --db production.db
```

```bash

loglens anomalies --window 30

# Limit results
loglens anomalies --limit 20

# Use custom database
loglens anomalies --db production.db
```
